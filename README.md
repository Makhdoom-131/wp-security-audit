🛡️ WP Security & Tech Scanner
A high-speed, frontend-driven WordPress security auditor. Scans for exposed endpoints, user enumeration vulnerabilities, and fingerprints active page builders/plugins.

✨ Features
Endpoint Auditing: Checks readme.html, wp-login.php, xmlrpc.php, and wp-json/wp/v2/users.
Tech Fingerprinting: Instantly detects Elementor, Divi, WPBakery, WooCommerce, and more.
Zero API Keys: Uses raw HTTP header logic via Cloudflare Workers. No third-party dependencies.
Cyber-Security UI: Sleek, terminal-inspired interface built with Vanilla JS.
🛠 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Cloudflare Workers (Serverless)
Architecture: GitHub Pages → Cloudflare Worker → Target Website
