
# WP Security Scanner

A high-speed WordPress security auditor. Scans for exposed endpoints and fingerprints active plugins.

## Features

- Endpoint Auditing: Checks readme.html, wp-login.php, xmlrpc.php, and REST API users.
- Tech Fingerprinting: Instantly detects Elementor, Divi, WPBakery, WooCommerce.
- Zero API Keys: Uses raw HTTP header logic via Cloudflare Workers.
